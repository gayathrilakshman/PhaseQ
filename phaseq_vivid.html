Modulation types — FFT explained
AM: Carrier height changes with message. FFT shows sidebands symmetrically around the carrier spike. Bandwidth = 2 × message frequency.
FM: Carrier frequency changes. FFT shows wider energy spread — multiple sidebands. Bandwidth ≈ 2(Δf + fm) by Carson's rule. More noise-resistant.
PM: Phase shifts at bit transitions. FFT similar to FM. Used in quantum — qubit state encoded as phase shift in microwave readout pulse.

Mixer math
sin(A) × sin(B) = ½cos(A−B) − ½cos(A+B). Two new frequencies always appear — sum and difference.

Homodyne: A = B so A−B = 0 (DC). The filter next keeps only this DC — which encodes phase. Rotate the phase slider and watch I change.

Heterodyne: A ≠ B so A−B = IF frequency. A new wave appears at the IF carrying both amplitude and phase.
